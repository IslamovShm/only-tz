# only-tz

Интерактивный таймлайн исторических дат с анимированным круговым навигатором.

## Стек

- React 19
- TypeScript 5
- Styled Components 6
- Swiper 12
- GSAP 3
- Webpack 5

## Установка

```bash
git clone https://github.com/IslamovShm/only-tz.git
cd only-tz


# Установить зависимости
npm install
```

## Запуск

### Режим разработки

```bash
npm start
```
